# MindShield AI

AI-powered mental wellness companion featuring facial emotion analysis, voice stress scanning, and empathetic AI chat.

## 🚀 Deploy to Glitch (100% Free)

1. **Import:** Go to [Glitch.com](https://glitch.com) -> **New Project** -> **Import from GitHub** -> Paste: `https://github.com/Ashwath1107/mindshield-ai-project1`
2. **Setup Env:** Open the `.env` file in the Glitch editor and paste your keys:
   - `DATABASE_URL`: (Your Neon PostgreSQL link)
   - `OPENROUTER_API_KEY`: (Your Gemini API key)
3. **Build:** Open the Glitch **Terminal** and run: `pnpm build`
4. **Done:** Glitch will automatically run the app. Click **Preview** to see it live!
